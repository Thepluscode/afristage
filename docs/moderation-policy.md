# Moderation Policy

v1 moderation priorities:

- Critical: child safety, sexual exploitation, credible violence.
- High: nudity, hate, harassment, fraud, scams.
- Medium: spam, impersonation, offensive behaviour.
- Low: off-topic or low-quality behaviour.

Required actions:

- Report user
- Report room
- Suspend room
- Suspend user
- Ban user
- Audit moderator actions

Moderation must exist before public launch.
